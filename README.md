## 📱 Deskripsi Aplikasi

Daily Notes App adalah aplikasi pencatatan sederhana berbasis React Native menggunakan Expo. Aplikasi ini memungkinkan pengguna untuk menambahkan, mengedit, menandai, dan menghapus catatan harian dengan tampilan yang modern dan interaktif. Dirancang untuk membantu pengguna mengelola aktivitas sehari-hari dengan lebih terstruktur.

---

## ✅ Fitur yang Diimplementasikan

* [x] Tambah catatan (Add)
* [x] Hapus catatan (Delete)
* [x] Edit catatan (Update)
* [x] Tandai catatan selesai (Mark as Done)
* [x] Filter catatan (Semua / Aktif / Selesai)
* [x] Counter jumlah karakter input
* [x] Validasi input (tidak boleh kosong)
* [x] List dinamis menggunakan FlatList
* [x] Empty state saat tidak ada data
* [x] Counter task selesai dari total task
* [x] UI responsif dengan Flexbox

## ▶️ Cara Menjalankan Project

1. Install dependencies:

```bash
npm install
```

2. Jalankan project:

```bash
npx expo start
```

3. Scan QR Code menggunakan aplikasi **Expo Go** di HP
